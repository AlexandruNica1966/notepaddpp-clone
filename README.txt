📦 Disponibil pentru Linux și Windows!

────────────────────────────────────────────────────────────────────────────────
 🐧 LINUX — AppImage (recomandat)
────────────────────────────────────────────────────────────────────────────────

  Fișier: Notepad++Qt-x86_64.AppImage (63 MB)

  1. Copiază AppImage-ul pe Desktop
  2. Setează permisiunea de execuție:

     chmod +x Notepad++Qt-x86_64.AppImage

  3. Rulează:

     ./Notepad++Qt-x86_64.AppImage

  4. (Opțional) Dublu-click direct din File Manager

  Cerințe: Linux x86-64, libfuse2 (prezent implicit)
  Dacă lipsește FUSE:  sudo apt install libfuse2

  Alternativ fără FUSE:
     ./Notepad++Qt-x86_64.AppImage --appimage-extract
     ./squashfs-root/AppRun

────────────────────────────────────────────────────────────────────────────────
 🪟 WINDOWS
────────────────────────────────────────────────────────────────────────────────

  Fișier: Notepad++Qt.exe (37 MB)

  1. Copiază Notepad++Qt.exe pe Desktop
  2. Dublu-click — pornește direct!
  3. Nu necesită instalare, nu necesită Python

  Cerințe: Windows 10/11 x64 (funcționează și pe Windows 7/8)
  Portabil: poți rula de pe USB stick, nu lasă urme în registry

 SHORTCUTS PRINCIPALE
────────────────────────────────────────────────────────────────────────────────

  Ctrl+N        — Fișier nou
  Ctrl+O        — Deschide fișier
  Ctrl+S        — Salvează

  Ctrl+W        — Închide tab
  Ctrl+Q        — Toggle comentariu linie
  Ctrl+F        — Find
  Ctrl+H        — Replace
  Ctrl+G        — Go to line
  Ctrl+Z / Ctrl+Y — Undo / Redo
  Ctrl+D        — Duplică linia
  Ctrl+L        — Șterge linia
  Ctrl+U        — lowercase
  Ctrl+Shift+U  — UPPERCASE
  Ctrl+F2       — Toggle bookmark
  F2 / Shift+F2 — Next / Previous bookmark
  F3 / Shift+F3 — Find next / previous
  F11           — Toggle tema dark/light
  Ctrl±         — Zoom
  Ctrl+0        — Reset zoom
